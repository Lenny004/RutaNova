# Fase 1 — Fundación del proyecto

## Objetivos

- Inicializar el repositorio, stack y estructura base de RutaNova.
- Configurar TypeScript, Tailwind v4, Prisma y variables de entorno.
- Dejar un README y convenciones listas para el equipo/agentes.

## Tareas

1. Crear estructura Next.js + TypeScript en la raíz.
2. Configurar Tailwind CSS v4 y tipografías del producto.
3. Definir `.env.example` con `DATABASE_URL`, `JWT_SECRET`, `APP_URL`.
4. Añadir scripts npm: `dev`, `build`, `start`, `db:migrate`, `db:seed`, `test`.
5. Documentar nombre, descripción y stack en `README.md`.

## Criterios de aceptación

- [ ] `npm install` y `npm run dev` levantan la app sin errores.
- [ ] Tailwind y tipografías cargan correctamente.
- [ ] `.env.example` y README están completos.
- [ ] Estructura de carpetas `src/app`, `src/components`, `src/lib`, `prisma` existe.
